App Store Connect → 앱 → (버전) → 한국어 현지화 항목에 그대로 붙여넣으세요. 괄호 안은 글자수 제한이며, 아래 초안은 모두 제한 이내입니다.

## 앱 이름 (30)
Notepad 메모장 — 텍스트 편집기

> "메모장" 단독은 이미 등록돼 있을 가능성이 큽니다. App Store Connect에서 예약한 이름에 맞춰 조정하세요.

## 부제 (30)
탭 · 세션 복원 · EUC-KR

## 프로모션 텍스트 (170)
Windows 메모장처럼 빠르고 가벼운 macOS 플레인 텍스트 편집기. 멀티 탭, 자동 세션 복원(저장 안 한 메모까지), 그리고 EUC-KR 등 한국어 인코딩을 든든하게 지원합니다.

## 설명 (4000)
Notepad는 Windows 11의 기본 메모장에서 영감을 받아 Swift로 새로 만든, 빠르고 군더더기 없는 macOS 플레인 텍스트 편집기입니다.

서식 없음. 마크다운 꾸밈 없음. AI 없음. 계정 없음. 오직 텍스트뿐 — 열고, 쓰고, 다음에도 그대로 있을 거라 믿으면 됩니다.

세션 자동 복원
탭을 열어둔 채 — 저장하지 않은 메모까지 — 앱을 종료해도, 다음 실행 때(재부팅 후에도) 떠난 그대로 모두 복원됩니다. 설정에서 매 실행 시 "이전 세션 계속"과 "새 세션 시작" 중 선택할 수 있습니다.

멀티 탭
한 창에서 여러 문서를 다룹니다. 드래그로 탭 순서를 바꾸고, Ctrl-Tab으로 순환하며, 저장하지 않은 탭에는 * 표시가 붙습니다.

한국어(그리고 레거시) 텍스트에 강함
UTF-8, UTF-8(BOM), EUC-KR, UTF-16(LE/BE)로 열고 저장합니다. 상태바에서 다른 인코딩으로 다시 열거나 현재 내용을 변환할 수 있고, 표현할 수 없는 문자가 있으면 분명하게 경고합니다. 한글 등 CJK 입력은 조합 중 손실 없이 안전하게 커밋됩니다.

필요한 건 다 있습니다
• 비모달 인라인 찾기/바꾸기 — 일치 개수, 둘러 찾기, 대소문자 옵션
• 인쇄 및 페이지 설정
• 줄로 이동, 시간/날짜 삽입, 자동 줄바꿈 토글, 확대/축소
• 다중 창
• 라이트/다크 모드

설계부터 프라이버시 우선
Notepad는 완전한 샌드박스에서 실행되며, 네트워크에 연결하지 않고, 광고가 없으며, 어떤 데이터도 수집하지 않습니다. 파일은 오롯이 당신의 Mac에 남습니다.
